Задание
Необходимо создать Ruby On Rails-приложение для хранения базы данных фильмов и предоставления REST API для работы с ними.

О фильме должна быть доступна следующая информация:
 
- Название на языке оригинала
- Локализованное название
- Год выпуска
- Рейтинг от 0 до 10
- Текстовое описание
- Изображение
- Страны производства
- Жанры

В API должны присутствовать следующие возможности:

- Создание фильма
- Редактирование фильма
- Удаление фильма
- Получение детальной информации о фильме
- Получение списка фильмов

Предоставить возможность отсортировать фильмы по одному из перечисленных полей:

- Год выпуска
- Рейтинг


Предоставить возможность отфильтровать фильмы по следующим полям:

- Название
- Год выпуска
- Страна производства
- Рейтинг
- Список жанров


Клиентскую часть реализовывать не нужно - только методы API.

В качестве СУБД используйте любую известную вам базу данных - от MySQL до MongoDB
